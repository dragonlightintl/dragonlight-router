# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/YOUR_USERNAME/dragonlight-router/security/advisories/new)
or email the maintainer directly.

**Expected response time:** within 72 hours for acknowledgement.

## Scope

- API key exposure or logging
- Authentication bypass in the HTTP server
- Privilege escalation via configuration loading
- Path traversal in cache file writes

## Out of Scope

- Vulnerabilities in upstream LLM provider APIs
- Issues requiring physical access to the machine running the router
